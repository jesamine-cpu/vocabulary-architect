# 單字建築師 (Vocabulary Architect)
AI 語音辨識輔助的英語單字補救學習平台

## 專案介紹 / Project Introduction
本專案為【我是AI創課師競賽 - 誠致程有威先生教育科技獎】參賽作品。

這是一個專為國中英語教學現場打造的「虛擬教室」與「互動演練場」。系統整合 Web Speech API 語音辨識技術，設計出「辨識→理解→語音輸出」的三層漸進式架構，幫助基礎落差較大的學生，能在無壓力的情境中進行刻意練習。透過即時的發音回饋與追蹤機制，有效支援學生自主學習，並輔助教師進行差異化教學。

## 系統需求 / Requirements
* 建議使用 Google Chrome 瀏覽器以獲得最佳的 Web Speech API 語音辨識支援。
* 需允許瀏覽器存取麥克風權限。
